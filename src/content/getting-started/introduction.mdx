# PostgreSQL이란?

**PostgreSQL**(포스트그레스큐엘)은 30년 이상의 역사를 가진 오픈소스 관계형 데이터베이스 관리 시스템(RDBMS)입니다.

## 왜 PostgreSQL인가?

PostgreSQL은 다음과 같은 이유로 전 세계 개발자들에게 사랑받고 있습니다:

- **ACID 완전 지원**: 데이터 무결성을 보장하는 트랜잭션 처리
- **확장성**: 사용자 정의 타입, 함수, 연산자 생성 가능
- **JSON/JSONB 지원**: NoSQL과 같은 유연한 데이터 모델링
- **전문 검색(Full-Text Search)**: 별도 엔진 없이 텍스트 검색 가능
- **풍부한 데이터 타입**: 배열, hstore, 범위 타입 등

## PostgreSQL의 주요 특징

### 표준 SQL 준수

PostgreSQL은 SQL 표준을 가장 충실히 구현한 데이터베이스 중 하나입니다. SQL:2023 표준의 대부분을 지원합니다.

### 버전별 주요 기능

| 버전 | 주요 기능 |
|------|-----------|
| PostgreSQL 13 | 증분 정렬, B-tree 중복 제거 |
| PostgreSQL 14 | 다중 범위 타입, JSON 편의 기능 |
| PostgreSQL 15 | MERGE 명령, 공개 스키마 권한 변경 |
| PostgreSQL 16 | 논리적 복제 개선, pg_stat_io |
| PostgreSQL 17 | 증분 백업, JSON_TABLE <VersionBadge version={17} /> |

## RDBMS 비교

```
PostgreSQL  ─  오픈소스, 기능 풍부, 확장성
MySQL       ─  오픈소스, 단순함, 빠른 읽기
SQLite      ─  내장형, 서버 불필요, 경량
Oracle      ─  상용, 엔터프라이즈, 고비용
```
