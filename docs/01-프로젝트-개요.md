# 후니프린팅 인쇄자동견적 시스템 — 프로젝트 개요

> **버전**: v1.0 | **작성일**: 2026-03-01 | **작성자**: 지니
> **분류**: 프로젝트 기획 문서 | **상태**: 초안 (프로젝트 리더 검토 전)

---

## 1. 프로젝트 목적

후니프린팅의 인쇄 상품을 온라인으로 자동 견적하고, 고객이 직접 주문할 수 있는 통합 시스템을 구축합니다.

**핵심 가치**
- 고객이 인쇄 옵션을 선택하면 즉시 자동으로 가격을 산출
- PDF 파일 업로드 또는 온라인 에디터(Edicus)로 주문 가능
- 주문 완료 시 MES(생산관리 시스템)로 자동으로 생산 정보 전달

## 3. 시스템 구성 요소 설명

### 3.1 Shopby (쇼핑몰 플랫폼)

Shopby Enterprise는 인쇄 상품 판매의 메인 플랫폼입니다. 고객은 Shopby 기반 쇼핑몰에서 상품을 조회하고 주문합니다.

- **역할**: 상품 목록 표시, 장바구니, 결제, 주문 관리
- **연동 방식**: Webhook 방식으로 주문 이벤트 수신
- **위젯빌더**: Shopby 스킨에 커스텀 위젯을 삽입하여 인쇄 옵션 선택 UI 제공

### 3.2 위젯빌더 (신규 개발)

Shopby 쇼핑몰 상품 상세 페이지에 탑재될 인쇄 옵션 선택 UI입니다.

- **기능**: 사이즈/인쇄/용지/후가공 옵션 선택 → 실시간 자동견적 → 장바구니 담기
- **주문 방식 선택**: PDF 파일 업로드 OR Edicus 에디터 실행
- **기술**: JavaScript/React 위젯 (Shopby 스킨 내 삽입)

### 3.3 PDF 파일 주문

고객이 직접 제작한 인쇄 파일(PDF, AI 등)을 업로드하여 주문하는 방식입니다.

- **대상 상품**: 대부분의 인쇄 상품
- **파일 규격**: 재단선 포함 PDF (블리드 3mm 포함)
- **검수**: 파일 자동 검수 또는 수동 검수

### 3.4 Edicus 에디터 주문

MotionOne의 Edicus 클라우드 에디터를 통해 고객이 온라인에서 직접 디자인하여 주문하는 방식입니다.

- **SDK**: Edicus JS SDK v2 (iframe 방식 에디터 삽입)
- **API**: Edicus Server API (api-dot-edicusbase.appspot.com)
- **기능**: 템플릿 선택, 텍스트/이미지 편집, 주문 연동
- **지원 상품**: 명함, 엽서, 스티커 등 규격 상품

### 3.5 MES (생산관리 시스템)

주문 완료 후 인쇄 생산에 필요한 정보를 처리하는 시스템입니다.

- **기술**: .NET (ASP.NET Core) WebAPI
- **운영 URL**: https://api.huniprinting.com
- **기능**: 생산 지시, 공정 관리, 재고 관리, 배송 처리
- **연동**: Shopby 주문 완료 → Webhook → 후니프린팅 API → MES REST API

## 4. 제품 현황

### 4.1 카테고리별 상품 수

| 카테고리 | 상품 수 | 특징 |
|----------|---------|------|
| 디지털인쇄 | 30개 | 출력비 + 용지비 + 후가공비 구조 |
| 스티커 | 12개 | 반칼/완칼/합판도무송 |
| 실사 (배너/현수막) | 28개 | 면적 기반 가격 |
| 아크릴 | 15개 | 크기 매트릭스 + 가공비 |
| 책자 | 9개 | 제본 방식별 단가 |
| 굿즈 (파우치/에코백 등) | 87개 | 고정단가 + 수량 할인 |
| 캘린더 | 5개 | 제본비 + 출력비 |
| 디자인캘린더 | 5개 | 에디터 기반 |
| 문구 | 1개 | 만년다이어리 |
| **합계** | **192개** | |

### 4.2 가격 계산 구조

```
최종 가격 = 출력비 + 용지비 + 후가공비 + 박/형압비 + 추가상품비

출력비: 수량 구간별 단가 × 판수 (양면은 단면의 2배)
용지비: (연당가 ÷ 2000) × 판걸이 수 × 로스율(1.3)
후가공비: 코팅 + 모서리 + 오시 + 미싱 + 가변 각각 독립 단가
박/형압비: 동판비(아연판, 1회성) + 박 가공비(수량별)
```
