# PostgreSQL 설치하기

<Callout type="tip">
이 사이트의 SQL 플레이그라운드는 PGlite를 사용하여 브라우저에서 직접 PostgreSQL을 실행합니다. 설치 없이도 학습을 시작할 수 있습니다.
</Callout>

로컬 환경에서 PostgreSQL을 사용하려면 아래 운영체제별 가이드를 따라 설치하세요.

## macOS

### Homebrew 사용

```bash
# Homebrew로 PostgreSQL 설치
brew install postgresql@16

# 서비스 시작
brew services start postgresql@16

# 버전 확인
psql --version
```

### Postgres.app 사용

[Postgres.app](https://postgresapp.com/)은 macOS용 PostgreSQL GUI 앱입니다. 다운로드 후 실행만 하면 됩니다.

## Linux (Ubuntu/Debian)

```bash
# 패키지 목록 업데이트
sudo apt update

# PostgreSQL 설치
sudo apt install postgresql postgresql-contrib

# 서비스 상태 확인
sudo systemctl status postgresql

# PostgreSQL 접속
sudo -u postgres psql
```

## Windows

1. [공식 다운로드 페이지](https://www.postgresql.org/download/windows/)에서 설치 파일 다운로드
2. 설치 마법사를 실행하여 설치
3. 설치 과정에서 비밀번호를 설정 (반드시 기억하세요)
4. pgAdmin이 함께 설치됩니다

## Docker로 설치

```bash
# PostgreSQL 16 컨테이너 실행
docker run --name pgstudy \
  -e POSTGRES_PASSWORD=mysecretpassword \
  -p 5432:5432 \
  -d postgres:16

# 컨테이너에 접속
docker exec -it pgstudy psql -U postgres
```

## 설치 확인

설치가 완료되면 터미널에서 다음 명령을 실행하여 확인합니다:

```sql
-- psql에 접속한 후
SELECT version();
```

다음과 같은 결과가 나타나면 성공입니다:

```
                          version
-----------------------------------------------------------
 PostgreSQL 16.x on ...
```

## 기본 데이터베이스 생성

학습용 데이터베이스를 생성합니다:

```sql
-- 새 데이터베이스 생성
CREATE DATABASE pgstudy;

-- 데이터베이스에 접속
\c pgstudy
```

<Callout type="info">
`psql`은 PostgreSQL의 공식 CLI 도구입니다. `\`로 시작하는 명령은 psql 메타 명령으로, SQL이 아닌 psql 전용 명령입니다.
</Callout>

## 유용한 psql 명령

| 명령 | 설명 |
|------|------|
| `\l` | 데이터베이스 목록 |
| `\dt` | 테이블 목록 |
| `\d 테이블명` | 테이블 구조 |
| `\q` | psql 종료 |
| `\?` | 도움말 |
