# 조건 필터링

`WHERE` 절을 사용하면 조건에 맞는 행만 조회할 수 있습니다.

## 기본 WHERE 절

```sql
SELECT * FROM products
WHERE price > 10000;
```

## 비교 연산자

| 연산자 | 설명 | 예시 |
|--------|------|------|
| `=` | 같음 | `WHERE status = 'active'` |
| `<>` 또는 `!=` | 다름 | `WHERE status <> 'deleted'` |
| `>`, `<` | 크다/작다 | `WHERE price > 1000` |
| `>=`, `<=` | 크거나 같다/작거나 같다 | `WHERE age >= 18` |

## 논리 연산자

### AND - 모든 조건 충족

```sql
SELECT * FROM products
WHERE category = '전자제품' AND price < 50000;
```

### OR - 하나 이상 충족

```sql
SELECT * FROM products
WHERE category = '식품' OR category = '음료';
```

### NOT - 부정

```sql
SELECT * FROM products
WHERE NOT (price > 100000);
```

## IN 연산자

여러 값 중 하나와 일치하는지 확인합니다:

```sql
SELECT * FROM products
WHERE category IN ('식품', '음료', '건강');

-- NOT IN
SELECT * FROM products
WHERE category NOT IN ('전자제품');
```

## BETWEEN 연산자

범위 내의 값을 찾습니다 (양쪽 끝값 포함):

```sql
SELECT * FROM products
WHERE price BETWEEN 10000 AND 100000;
```

## LIKE 패턴 매칭

| 와일드카드 | 설명 |
|-----------|------|
| `%` | 0개 이상의 임의 문자 |
| `_` | 정확히 1개의 임의 문자 |

```sql
-- '노'로 시작하는 이름
SELECT * FROM products WHERE name LIKE '노%';

-- '드'로 끝나는 이름
SELECT * FROM products WHERE name LIKE '%드';

-- 두 번째 글자가 '우'인 이름
SELECT * FROM products WHERE name LIKE '_우%';
```

<Callout type="tip">
대소문자를 구분하지 않으려면 `ILIKE`를 사용하세요. 이는 PostgreSQL 전용 기능입니다.
</Callout>

## NULL 체크

```sql
-- NULL인 행 조회
SELECT * FROM products WHERE stock IS NULL;

-- NULL이 아닌 행 조회
SELECT * FROM products WHERE stock IS NOT NULL;
```

<Callout type="warning">
`WHERE stock = NULL`은 동작하지 않습니다. NULL 비교는 반드시 `IS NULL` / `IS NOT NULL`을 사용하세요.
</Callout>

## 복합 조건

괄호로 조건의 우선순위를 명확히 합니다:

```sql
SELECT * FROM products
WHERE (category = '전자제품' OR category = '음료')
  AND price < 50000
  AND stock > 0;
```
