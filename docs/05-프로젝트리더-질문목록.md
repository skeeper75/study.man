## 4. 가격 계산 정책

### 🟡 Q4-1. 가격 계산 공식 최종 확인

> 현재 파악한 가격 계산 공식이 맞는지 확인이 필요합니다.

**현재 파악된 공식**:
```
최종 가격 = 출력비 + 용지비 + 후가공비 + 박/형압비 + 추가상품비
출력비 = 수량구간별 단가 × 판수 (양면은 단면의 2배)
용지비 = (연당가 ÷ 2000) × 판걸이수 × 로스율(1.3)
후가공비 = 각 후가공 독립 단가의 합계
```

**질문**: 이 공식이 모든 카테고리에 동일하게 적용되나요?
실사/배너(면적 기반), 아크릴(크기 매트릭스), 굿즈(고정단가)는 별도 공식인가요?
