# 첫 번째 쿼리

SQL(Structured Query Language)은 데이터베이스와 대화하는 언어입니다. 가장 기본적인 SQL 명령인 `SELECT`부터 시작해봅시다.

## SELECT로 데이터 조회하기

`SELECT`는 데이터를 조회하는 명령입니다. 가장 간단한 형태로 시작합니다:

```sql
SELECT 1 + 1 AS result;
```

## 문자열 다루기

PostgreSQL에서 문자열은 작은따옴표(`'`)로 감쌉니다:

```sql
SELECT 'Hello, PostgreSQL!' AS greeting;
```

<SQLPlayground defaultQuery="SELECT 'Hello, PostgreSQL!' AS greeting;" title="문자열 출력" />

문자열을 합치려면 `||` 연산자를 사용합니다:

```sql
SELECT 'Hello' || ', ' || 'World!' AS message;
```

## 여러 값 동시에 조회

```sql
SELECT
  42 AS answer,
  3.14 AS pi,
  'PostgreSQL' AS database_name,
  NOW() AS current_time;
```

## 테이블에서 데이터 조회하기

실제 데이터베이스에서는 테이블에 저장된 데이터를 조회합니다. 먼저 간단한 테이블을 만들고 데이터를 넣어봅시다:

```sql
-- 테이블 생성
CREATE TABLE fruits (
  id SERIAL PRIMARY KEY,
  name TEXT NOT NULL,
  color TEXT,
  price INTEGER
);

-- 데이터 삽입
INSERT INTO fruits (name, color, price) VALUES
  ('사과', '빨강', 1500),
  ('바나나', '노랑', 3000),
  ('포도', '보라', 5000),
  ('딸기', '빨강', 8000),
  ('오렌지', '주황', 2000);

-- 전체 데이터 조회
SELECT * FROM fruits;
```

## 특정 컬럼만 조회

```sql
SELECT name, price FROM fruits;
```

## 별칭(Alias) 사용

`AS` 키워드로 컬럼에 별칭을 붙일 수 있습니다:

```sql
SELECT
  name AS "과일 이름",
  price AS "가격(원)"
FROM fruits;
```

<Callout type="tip">
한글이나 공백이 포함된 별칭은 큰따옴표(`"`)로 감싸야 합니다.
</Callout>

## SQL 작성 규칙

1. **키워드는 대소문자를 구분하지 않음**: `SELECT`, `select`, `Select` 모두 동일
2. **세미콜론으로 끝남**: 각 SQL 문은 `;`으로 마무리
3. **공백과 줄바꿈**: 가독성을 위해 자유롭게 사용 가능
4. **문자열은 작은따옴표**: `'Hello'` (큰따옴표는 식별자용)
